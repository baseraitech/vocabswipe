# VocabSwipe Landing Page

This folder contains a ready-to-use static landing page for the VocabSwipe app.

## Files

- `index.html` — Main landing page with app hero, features, screenshots, and CTA.
- `privacy.html` — Privacy Policy page.
- `terms.html` — Terms of Use page.
- `styles.css` — Styling for the website.
- `script.js` — Mobile navigation toggle.

## How to preview

Open `vs_web/index.html` in a browser, or serve the folder with a local static server.

If you want the screenshots and logo to load correctly, keep the image files in the `vs_web/assets` folder:

- `vs_web/assets/vocabswipe-newturuncu.png`
- `vs_web/assets/vs-ss-6.jpg`
- `vs_web/assets/vs-ss-7.jpg`
